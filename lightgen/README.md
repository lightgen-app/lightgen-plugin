# LightGen

Lightweight MCP that brings image creation to Claude.

## Install

```
/plugin marketplace add lightgen/lightgen-plugin
/plugin install lightgen
```

Or via CLI:

```
npx @lightgen/cli setup
```

## Usage

Once installed, just ask Claude to generate an image:

> "Generate an image of a neon sign that says HELLO on a dark brick wall"

Or use the skills:

- `/lightgen:generate` — Generate an image
- `/lightgen:balance` — Check remaining images
- `/lightgen:setup` — Purchase images and configure

## How it works

LightGen is a lightweight MCP plugin that connects Claude to high-quality image generation. One plugin, zero bloat — just ask Claude to make an image.
